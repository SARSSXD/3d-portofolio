# 3D Developer Portfolio - Kautsar Quraisy Al Hamidy

Welcome to my 3D Developer Portfolio website! This is an interactive, fully responsive 3D web application designed to showcase my skills, experience, and projects as a Fullstack Engineer.

The application features premium 3D graphics, rich animations, and a modern sleek dark UI.

## 🚀 Tech Stack

This project is built using modern front-end technologies:

- **Core Framework**: [React.js](https://reactjs.org/) (Vite bundler)
- **3D Graphics & Rendering**:
  - [Three.js](https://threejs.org/) (Core 3D library)
  - [@react-three/fiber](https://github.com/pmndrs/react-three-fiber) (React wrapper for Three.js)
  - [@react-three/drei](https://github.com/pmndrs/drei) (Useful helpers for React Three Fiber)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) (Declarative page and element transitions)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Utility-first CSS framework)
- **Email Service**: [@emailjs/browser](https://www.emailjs.com/) (Direct email transmission from contact form)

---

## ✨ Features

- **Interactive 3D Scene**: Includes an interactive 3D Computer/Desktop model, a floating 3D Earth/Planet, and dynamic star particles floating in the background.
- **Dynamic Projects Showcase**: Visual cards showcasing my work and links to live demos or source repositories.
- **Interactive Timeline**: Vertical progress timeline illustrating my professional work experience.
- **Responsive Layout**: Designed with a mobile-first approach, fully optimized for desktops, tablets, and mobile screens.
- **Fully Working Contact Form**: Sends messages directly to my inbox via EmailJS.

## 🛠️ Local Development

Follow these steps to set up and run the project locally on your machine.

### Prerequisites

Make sure you have Node.js installed (v18+ recommended) and npm.

### Installation

1. Clone or copy this repository to your local directory:
   ```bash
   cd project_3D_developer_portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the local development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

### Building for Production

To create an optimized production build:

```bash
npm run build
```

This will output static files into the `dist/` directory, which can be deployed to any static site host (Vercel, Netlify, GitHub Pages, etc.).
